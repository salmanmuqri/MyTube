# MyTube Agent Handoff Status

Last updated: 2026-03-15

## Project Snapshot

- Stack: Django + DRF backend, React + Vite frontend, Nginx reverse proxy, PostgreSQL, Redis, Celery worker/beat
- Runtime: Docker Compose
- Theme: NightStream (indigo/slate dark UI)
- Streaming: HLS generated by FFmpeg and served from `/media/`

## Core Architecture Map

- Backend apps:
- `users`: auth/profile/subscriptions
- `videos`: upload, processing, playback metadata, playlists, search, likes/comments
- `recommendations`: recommendation model and tasks
- `analytics_app`: watch history and analytics

- Frontend major areas:
- pages for home/watch/upload/profile/trending/history/channel/subscriptions/playlists
- custom `VideoPlayer` for HLS playback and controls
- API layer in `frontend/src/api/services.js`

## Feature Status

Completed:

Completed recently (stability):
- enabled `django_celery_beat` in Django apps to match beat scheduler
- fixed Celery beat startup errors
- expanded `Video.original_file` and `Video.thumbnail` path max length to prevent long filename save errors
- set upload in-memory threshold to 10 MB so multi-GB files are streamed to disk temp
- increased backend and proxy request timeouts for very large uploads

## Important Config Values

- Nginx max upload: `client_max_body_size 5120M`
- Django request limit: `DATA_UPLOAD_MAX_MEMORY_SIZE = 5 GB`
- Django in-memory file threshold: `FILE_UPLOAD_MAX_MEMORY_SIZE = 10 MB`
- Gunicorn: 3 workers, timeout 1800s
- Nginx API proxy: buffering off, read/send timeout 1800s

## Known Operational Risks

- Very large transcodes are CPU-intensive and may take significant time.
- Celery concurrency and host resources directly affect processing throughput.
- Original files are deleted post-success; failed jobs may require re-upload.

## Quick Verification Commands

- service status:
- `docker compose ps`

- backend logs:
- `docker compose logs backend --tail=200`

- celery logs:
- `docker compose logs celery_worker --tail=200`
- `docker compose logs celery_beat --tail=200`

- migrations:
- `docker compose exec backend python manage.py showmigrations videos django_celery_beat`
